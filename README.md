# Mintlify Website Clone

This project is a simple front-end clone of the Mintlify website, created using **HTML** and **CSS**.  
It was built as part of an assignment to practice UI layout, styling, and responsive design.

## Features
- Clean and modern UI inspired by Mintlify
- Responsive layout
- Custom fonts and colors
- Hover effects and smooth styling
- Pure HTML & CSS (no frameworks)

## Tech Stack
- HTML5
- CSS3

## Project Structure
├── index.html

├── style.css

└── images/


## How to Run
1. Download or clone the project
2. Open `index.html` in any modern web browser
3. That’s it — no setup required

## Purpose
- Practice front-end development
- Understand layout structure and styling
- Improve CSS skills by recreating a real-world UI
